# pyconkr2021-APITest
PyCon Kor 2021 API Response Test Example w/ PyTest & Schema

## Requirements
- Python 3.8+
- Docker
- Docker Compose

## Run PyTest
Clone this repository:
```
git clone https://github.com/qu3vipon/pyconkr2021-APITest.git
```

Run PyTest
```
docker-compose up --build --exit-code-from app
```
